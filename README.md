### Overview

This repository documents my Python learning journey, from fundamental programming concepts to algorithms, data structures, object-oriented programming, and larger practice projects.
The work in this repository was developed as part of my ongoing coding practice, following my early experience with Scratch and LiveCode.  **05. Lists, Dictionaries & Sets:** Covers important Python data structures and how they are used to store, organise, and manipulate data.

 **07. Object-Oriented Code:** Introduces classes, objects, attributes, methods, and object-oriented programming structure.
 - **Algorithms:** Contains algorithm-focused files, including logic-based and mathematical problem-solving programs.
